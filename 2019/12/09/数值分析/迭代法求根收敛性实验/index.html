迭代法求根收敛性实验

categories: 数值分析

实验要求

求下列方程的实根：
$$x^2-3x+2-e^x=0$$
$$x^3+2x^2+10x-20=0$$
要求：（1）设计一种不动点迭代法，要使迭代序列收敛，然后再用斯特芬森加速迭代，计算到$|x_k-x_{k-1} |<10^{-8}$为止。（2）用牛顿迭代，同样计算到$|x_k-x_{k-1} |<10^{-8}$为止。输出迭代初值及各次迭代值和迭代次数k，比较方法的优劣。

运行环境

采用matlab2019a中的live script实时脚本运行

实验过程

见github页面内实时脚本的内容

实验结果

迭代速度: 斯特芬森加速迭代,牛顿迭代>不动点法

斯特芬森加速迭代与牛顿迭代的好坏不能简单下结论,事实上斯特芬森迭代依赖于不动点法的迭代公式的好坏.而且不如不动点法与牛顿法稳定,甚至在实验时发生混沌现象

不同方法在不同初值选取下迭代次数探究

方程1

$$x^2-3x+2-e^x=0$$

不动点迭代公式:$x=\frac{x^2+2-e^x}3$

不动点迭代公式:$x=\frac{e^x-2}{x-3}$

可以看出史蒂芬森在-50-0范围内剧烈震荡,然后x0<-50时看出史蒂芬森比牛顿法更好,但是在大于0时,该方法反而不如牛顿法.更有趣的是,加速法发挥在-50,0范围内反而更加稳定

方程2

$$x^3+2x^2+10x-20=0$$

不动点迭代公式:$x=\frac{20}{x^2+2x+10}$

可以看出newton法胜在稳定,但当迭代公式选择恰当时,史蒂芬森能够迭代地非常快,甚至超越牛顿法.

函数

function r=BuDongDieDai(DieDaiFun,x0)
r=inf;
k=0;%迭代次数
while abs(r-x0)>1e-8
     r=x0;
     y=DieDaiFun(x0);
     x0=y;
     fprintf('\n次数k=%d;方程的根x=%.11f;f(x)=%.8f;误差为%f\n',k,r,y,abs(r-x0));%保留11位小数
     k=k+1;

end 
end

function r=newton(f,df,x0)
r=inf;
k=0;%迭代次数
while abs(r-x0)>1e-8
     r=x0;
     y=f(x0);
     x0=x0-f(x0)/df(x0);
     fprintf('\n次数k=%d;方程的根x=%.11f;f(x)=%.8f;误差为%f\n',k,r,y,abs(r-x0));%保留11位小数
     k=k+1;
     abs(r-x0)
end 
end

function[x_star,index,it]=steffensen(phi,x,ep)
%斯特芬森加速迭代方法
% x为初始点
% x_star为当迭代成功时，输出方程的根
%   当迭代失败时，输出最后的迭代值;
% index为指标变量，当index=1时，表明迭代成功
% it为迭代次数
index=0;k=1;
fprintf('\n次数k=0;方程的根x=%.11f;f(x)=%.11f\n',x,phi(x));
while k< 500
    x1=x;y=feval(phi,x);z=feval(phi,y);
    x=x-(y-x)^2/(z-2*y+x);

    fprintf('\n次数k=%d;方程的根x=%.11f;f(x)=%.8f;误差为%f\n',k,x,y,x-x1);%保留11位小数
    if abs(x-x1)<ep
        index=1;break;
    end
    k=k+1;
end
x_star=x;it=k;
end
