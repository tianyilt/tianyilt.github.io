# shell脚本
categories: Linux
## 基础知识

1. 第一行声明shell的名称

   ```
   #!/bin/bash
   ```

2. 程序结构

   * 声明shell名称
   * 程序内容:
     * 内容与功能
     * 版本信息
     * 作者联系方式
     * 创建日期
     * 维护记录
   * 声明环境变量, PATH是最重要的, 让程序运行的时候可以直接下达指令
   * 主要程序
   * 执行成果告知, `exit 0`表示离开程序, 返回一个0给系统. 

3. 程序运行

   * 保存为`.sh`格式

   * 使用`./filename.sh`或者`sh filename.sh`执行

   * 注意要有`x`权限
